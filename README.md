# README.md

This repo hosts an online survey for COMP 3008 Assignment 3 (Question 2).

Survey title: **Jiejia Zhou's Questionnaire**

## Purpose
This survey evaluates perceived usability of the Expedia mobile app for the task:
- Find a 2-night hotel stay in Niagara Falls for 2 guests
- Identify the **total price including taxes and fees** before booking

The survey includes:
1) **SUS (System Usability Scale)** (10 closed-ended items)  
2) A **task-specific questionnaire** (7+ closed-ended items)

## How to deploy (Netlify)
1. Upload these files to a GitHub repo:
   - `index.html`
   - `success.html`
   - `README.md`
2. Connect the GitHub repo to Netlify
3. Deploy the site
4. Copy your Netlify URL and submit it for Q2(b)

## Notes
- All questions are **closed-ended**.
- The task-specific questions use consistent rating scales to make responses easier to analyze.
- The form uses Netlify Forms and redirects to `success.html` after submission.
- No names are collected (privacy-friendly).# user-interface-survey
